# Python Analysis Scripts

Data analysis for DESI and Pantheon+ supernova datasets.

| Script | Description |
|--------|-------------|
| `analisi_DES_coordinate.py` | DESI data in coordinate space |
| `analisi_DES_v2.py` | Updated DESI analysis |
| `analisi_quadrupolo_DES.py` | Quadrupole extraction from DESI |
| `analisi_quadrupolo_pantheon.py` | Quadrupole from Pantheon+ |
| `analisi_quadrupolo_pantheon_v2.py` | Updated Pantheon+ analysis |

Key result: w(z) crossing w=−1 compatible with DESI at 0.7σ [N]

# Geometry Scripts

Mathematica scripts for the MP-AdS5 metric, horizon conditions, and S/D/R channel structure.

| Script | Description |
|--------|-------------|
| `NEC_esatta.wls` | Null Energy Condition verification for GLPP metric |
| `dDda_GPP.wls` | Derivative of horizon area with respect to spin parameters |
| `g4_completo.wls` | Full induced 4D metric on the brane |
| `g4_scaling.wls` | Scaling analysis of the induced metric |
| `su3_modi_S3.wls` | SU(3) modes on squashed S³, H₂ topology analysis |
| `kk_gauge_GLPP.wls` | Kaluza-Klein gauge modes on GLPP background |
